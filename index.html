このテンプレートについて

1ファイルのHTMLと1ファイルのCSSだけで構成されたミニマルな構成です。 JavaScriptを使わずに、シンプルな構成で「それっぽい」見た目を実現しています。

テキストとカラー、画像（必要なら）を差し替えるだけで、個人サイトやサービスLPとしてすぐ公開できます。

必要スキル HTML / CSS
